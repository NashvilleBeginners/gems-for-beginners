7 Gems You'll Meet as a Rails Dev

  • Devise
  • CanCan
  • Ransack
  • Draper
  • Will Paginate
  • Slim
  • Simple Form

Devise

Purpose: User authentication

Notable Features: current_user, views for sign up/sign in/sign out, password recovery/verification via email

Alternative: Sorcery

CanCanCan

Purpose: User authorization

Notable Features: - if can?(do: Thing), load_and_authorize_resource

Alternative: Pundit

Ransack

Purpose: Searching and Sorting

Notable Features: search_form_for, search association attributes, sort_link

Alternative: MetaSearch*, Sunspot*

Draper

Purpose: Easy-to-use decorators

Notable Features: decorate/decorate_collection, decorates_assigned, decorates_association

Alternative: DIY decorators

Will Paginate

Purpose: pagination

Notable Features: Model.paginate, will_paginate @model

Alternative: Kaminari

Slim

Purpose: less cranky template syntax

Notable Features: no closing tags, no <%= syntax

Alternative: ERB, HAML

SimpleForm

Purpose: simpler form building

Notable Features: automatic labels, build associations, well-supported, easily customized

Alternative: Formtastic, Rails forms

Responders

Purpose: DRY controllers

Notable Features: dramatically shorter controller code

Alternative: n/a

Pry

Purpose: debugging tool

Notable Features: navigate source code, run-time console, whereami

Alternative: IRB
